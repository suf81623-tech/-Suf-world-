# suf world

A modern social media web app built with HTML, CSS, and JavaScript.

## Features

✔ User profile with username and profile picture  
✔ Create posts with images  
✔ Like posts  
✔ Comment and reply to comments  
✔ Edit and delete posts  
✔ Dark mode  
✔ Cloudinary for image storage  
✔ JSONBin for database storage  
✔ Responsive and professional UI

## Technologies

- HTML
- CSS
- JavaScript
- Cloudinary (image hosting)
- JSONBin (database)

## Setup

1. Create account on JSONBin  
2. Create new bin and copy BIN ID and API key  
3. Use Cloudinary account and upload preset  
4. Replace credentials in code  
5. Open in browser

## Usage

- Upload profile picture  
- Create posts  
- Like and comment  
- Edit or delete posts  
- Switch dark mode
